# Promotion Impact Analysis on E-Commerce Sales

This project analyzes the effectiveness of promotional campaigns on Amazon sales using statistical analysis and machine learning models.

## Dataset
~128K Amazon transaction records

## Key Analysis
- Data preprocessing and feature engineering
- Statistical hypothesis testing (ANOVA, Chi-Square, Correlation)
- Predictive modeling using Random Forest and Gradient Boosting
- Sales trend and promotional impact evaluation

## Results
Random Forest achieved highest predictive accuracy with R² ≈ 0.85.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Statistics, Machine Learning
